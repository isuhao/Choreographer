Detailed Description

Describes a point in space at a certain time.

Used by AudioSourceAmbipanning::spacialEnvelope, by AudioSourceAmbipanning::newSpacialEnvelope and by the argument newSpacialEnvelope of AudioSourceAmbipanning::setSpacialEnvelope.
