General


Project Setup

First, select a folder for your project. Please save all your project-files (*.hkp) in this folder. If you need to make a copy, please use the finder to copy the entire project folder to a new location. Hagakure will copy all the mono-soundfiles added to the project to the automatically generated folder 'Audio' in this location.

Next open the Patcher (Command-Key + B) and select the desired audio-device and a rig. See also the patcher help.

To select the samplerate, open the Project-Settings-Dialog (either from the Project menu or Command-Key + . (dot).

Open the Pool (Command-Key + P) and load mono files to the pool (via the context menu or Command-Key + L). Once added, drag files to the arrange window. See also the pool help.
