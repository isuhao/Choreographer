Keyboard shortcuts
Space Start / Stop playing
0 Stop playing
Enter Start playing
+ Pause playing
. Set cursor position to 0.
1 Move the cursor to the position indicated by the left locator
2 Move the cursor to the position indicated by the right locator
Control-Key + 3 - 9 Set the respective marker
3 - 9 Set the cursor to the position of the respective marker
L Loop on / off
S SnapsToGrid on / off
F Follow on / off


Command-Key + T shows or hides the Transport-Bar.

Copying audioevents
Hold the Command-Key and drag the selected audioevent(s) to the new position.

Command-Key + Left Click in Ruler view:
Sets the right locator to the position of the mouse click, or the nearest possible position if SnapsToGrid is enabled.

Alternate-Key + Left Click in Ruler view:
Sets the left locator to the position of the mouse click, or the nearest possible position if SnapsToGrid is enabled.
